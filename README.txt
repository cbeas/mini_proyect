

App to manage newsletter subscribers:
	-Display list of subscribers
	-Import subscribers from csv 
	-Export subscribers to csv 

Install instructions:

Please install email_validator:
	pip install email_validator

Python version: 3.6.8
Django version: 2.2.6
SQLite version: 3.22.0 

Run proyect:
	python manage.py runserver "your_ip":"your_port"


URL: /admin/subscribers	


Notes:
	For demonstration purposes I use SQLite for a production server we should use another database backend
